# Theme Selection Guide

Decision matrix for choosing the right theme based on content and context.

---

## Quick Decision Tree

```
Is it a tech/programming talk?
  YES → tech
  NO ↓

Is it corporate/business?
  YES → business
  NO ↓

Does the audience prefer dark mode?
  YES → dark
  NO ↓

Is it creative/event/youth-oriented?
  YES → colorful or gradient
  NO ↓

Is it academic/data-heavy/research?
  YES → minimal
  NO ↓

Use → default
```

---

## Theme Matrix

| Theme | Background | Best For | Avoid When |
|-------|-----------|----------|------------|
| **default** | Warm beige | General talks, lectures, seminars | Very technical or very creative content |
| **minimal** | White | Academic, research, data-focused | Needs visual excitement or brand presence |
| **colorful** | Pink/purple gradient | Events, hackathons, creative pitches | Formal settings, readability-critical content |
| **dark** | Dark navy | Tech talks, evening events, modern brand | Printed handouts, bright-room projectors |
| **gradient** | Shifting gradients | Visual storytelling, keynotes, launches | Data-heavy content, lots of tables |
| **tech** | GitHub dark | Dev talks, tutorials, code walkthroughs | Non-technical audiences |
| **business** | White + navy border | Board meetings, proposals, reports | Informal or creative contexts |

## Theme by Audience

| Audience | Primary Theme | Alternative |
|----------|--------------|-------------|
| Developers | tech | dark |
| Executives | business | minimal |
| Students | colorful | default |
| Researchers | minimal | default |
| Designers | gradient | colorful |
| General public | default | minimal |
| Startups/pitches | gradient | colorful |

---

## Theme by Content Type

| Content | Theme | Why |
|---------|-------|-----|
| Code-heavy | tech | Monospace font, syntax colors, dark bg |
| Data/tables | minimal or business | Clean table styling, no visual noise |
| Quotes/stories | gradient | Each slide feels distinct |
| Step-by-step tutorial | dark or tech | Good code contrast, numbered sections |
| Sales/proposal | business | Professional, structured, trustworthy |
| Conference keynote | gradient or colorful | High visual impact |
| Internal training | default | Neutral, focused on content |
